│   ├── 무형자산 (영업권 / 산업재산권 / 소프트웨어 / 기타무형자산)

│   ├── 매입채무및기타채무 (미지급금 / 미지급비용 / 기타채무)

│   ├── 기타충당부채
│   └── 기타유동부채 (선수금 / 선수수익 / 예수금 / 부가세예수금/대급금)

    ├── 장기매입채무및기타채무

### [4.포괄손익계산서] — IFRS P/L + OCI 전체 계정

**수익 분류 (영업수익 → 플랫폼별 → 상품별)**:
- 광고: 디스플레이(CPM) / 쇼핑(CPM) / 비즈니스플랫폼(CPC/CPS)
- IT플랫폼: 네이버페이(결제/채널수수료) / IT서비스 / 클라우드
- 콘텐츠서비스: 게임 / 음원 / 웹툰 / V LIVE
- 라인 및 기타 플랫폼: 라인 / 스노우

**비용 분류**:
- 플랫폼 개발/운영: 인건비(급여/상여/퇴직급여/복리후생) / 지급임차료 / 세금공과금 / 소모품비 / 접대비 / 여비교통비
- 대행/파트너: 결제수수료 / 판매수수료 / 영업대행 / 컨텐츠수수료 / 금융수수료 / 라이선스
- 인프라비: 인건비 / CDN / 상면비 / 인터넷백본 / 감가상각비(IDC/서버/네트워크)
- 마케팅비: 매체비(TV/인터넷/인쇄) / 프로모션 / 홍보비 / 협찬비

**영업외 손익**:
- 기타수익: 수입임대료 / 외환차익 / **외화환산이익** / 유무형자산처분이익 / 종속기업투자처분이익 / **지분법적용투자주식처분이익** / 대손충당금환입
- 기타비용: 외환차손 / **외화환산손실** / 유무형자산처분손실/손상 / 기부금 / 종속기업투자처분손실 / **지분법적용투자주식손상차손**
- 금융수익: 이자수익 / 배당금수익 / 외환차익(금융) / 외화환산이익(금융) / **당기손익공정가치측정금융자산평가이익** / **파생금융상품거래이익** / **매도가능금융자산처분이익** / **기타포괄손익공정가치측정금융자산처분이익-채무상품** / 상각후원가측정금융상품처분/손상차손환입
- 금융비용: 이자비용 / 금융수수료 / 외환차손(금융) / 외화환산손실(금융) / **당기손익공정가치측정금융자산평가손실** / **파생금융상품거래손실** / 사채상환손실
- **지분법손익**: 지분법평가이익 / 지분법평가손실 (관계기업)
- 법인세비용
- 중단사업손익
- 당기순이익

## 5. 결론 요약

**기존 아키텍처가 잘 커버하는 영역**: 복식부기 핵심, 계정 계층, 외환차손익(실현), 결산 5단계, 세무조정, FiscalPeriod, 이익잉여금

**즉시 보완 필요한 영역**:
